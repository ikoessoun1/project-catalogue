  1. TOPIC 1: CSS SELECTORS AND PSEUDO CLASSES

    SELECTORS

    What are selectors? Selectors are that which point the browsers to the elements to be styled or for the css rule to apply to.

    Examples of styles are:

    * {background-color: antiquewhite; border: solid red 10px;} In the code, * is the selector telling the browser what elements are to be styled. "*" represents the universal selector which means to style all elements so.
